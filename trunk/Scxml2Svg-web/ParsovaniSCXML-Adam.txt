  <chapter>
    <title>Zpráva o práci na projektu scxml2svg-web</title>
    <para>
		Mým úkolem bylo extrahovat stavy a přechody z SCXML souboru do příslušných objektů. Pro parsování zvoleného SCXML dokumentu používám DOM parser DocumentBuilder z Java API for XML Processing (JAXP). 
	</para>
	<para>	
		Pro dokument nejprve vytvořím objekt File a ten potom předám objektu třídy DocumentBuilder, která jej zvaliduje pomocí XML schématu dostupného na http://www.w3.org/2011/04/SCXML/scxml.xsd. Tímto se ověří, jestli zpracovávaný dokument je validní SCXML dokument. 
	</para>
	<para>	
		Pokud validace projde bez problému, vytvořím pro každý stav z SCXML dokumentu objekt State. Uložím také jeho atributy. Zpracuji i finální stavy (final states), paralelní stavy (parallel states) a history stavy. Do každého stavu také uložím odkaz na jeho předka a odkaz na všechny jeho potomky.
	</para>
	<para>	
		Potom podobně jako stavy uložím přechody mezi stavy do objektů třídy Transition. Jako atributy těchto objektů uložím především zdrojový a cílový stav. Ukládám také iniciální přechody (XML elementy transition, které mají za předka element &lt;initial/&gt;).
	</para>
	<para>	
		Všechno provádím ve třídě SCXMLParser. Po úspěšném zpracování SCXML souboru už stačí pouze tuto třídu požádat například o list všech stavů bez předka (metoda getRootStates()) nebo o list všech přechodů mezi stavy (getAllTransitions).
	</para>
  </chapter>
